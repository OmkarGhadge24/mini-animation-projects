### Repository structure

Each project follows the same simple structure:

```txt
Project-N/
  index.html        # Markup and script tags (GSAP, plugins)
  styles.css        # Layout and styling
  script.js         # Animation logic
  assets/           # Images, SVGs, fonts (optional)
```

## Run Locally
1. **Clone the project**

  ```bash
    git clone https://github.com/OmkarGhadge24/mini-animation-projects.git
  ```

2. **Go to the project directory**

  ```bash
    cd mini-animation-projects
  ```

3. **Navigate to any project folder**

  ```bash
    cd Project-N
  ```

4. **Open with Live Server**
   
   - Install the "Live Server" extension in VS Code

   - Or click the "Go Live" button in VS Code status bar
